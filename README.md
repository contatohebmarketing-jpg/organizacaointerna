# Organização Interna — HEB Marketing

Fonte de verdade (versionada) do sistema operacional da HEB. Espelha o
**Painel de Comando T3** no Notion e serve de base para processualizar a operação.

## A ordem é obrigatória

> **1. Oferta → 2. Entrega → 3. Operação.**
> A ferramenta é a *última* camada. O caos vem de tentar montar a camada de
> cima (app/automação) antes de fechar a de baixo (oferta). Foi o que
> aconteceu com o Asana, montado e largado.

Gestão = **garantir que as coisas aconteçam.** Resultado do líder = resultado
do time = resultado da empresa.

1. **Fase 1 — Fundação de gestão (em andamento):** ritos + metas por pessoa.
2. **Fase 2 — Sair do gargalo-raiz:** processualizar Comercial e ASCEND
   (social media já tem forma). Usar transcrições de reunião → processos.
3. **Fase 3 — Ferramentas/automações a serviço do processo:** CRM no Notion,
   webhook respondi.app → Notion, enciclopédia de prompts.
